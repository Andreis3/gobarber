### Configuração da Aplicação

#### yarn ou npm install

    -para instalar dependências

## Execução dos testes

 #### yarn test:all

    -executa todos os testes integration e unit

 #### yarn test:unit

    -executa somente os testes unitarios

 #### yarn test:integration

    -executa somente os testes de integração
